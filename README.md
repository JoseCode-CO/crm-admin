

## Importante

Clonar Proyecto, instalar dependencias, configurar db.

Ejecutar los siguientes comandos:

## npm run dev: 
Compilar estilos 

## php artisan serve: 
Iniciar servidor.

## php artisan migrate: 
Correr migraciones.

## php artisan db:seed: 
Para llenar la base de datos con datos de prueba.
